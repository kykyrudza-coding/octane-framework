# Release 0.2.0

Version `0.2.0` додає два великі блоки framework API: Halcyon ORM і DTO, а також переводить application skeleton на config-driven bootstrap.

## Added

- Halcyon ORM package:
  - model metadata parser/repository;
  - metadata cache;
  - hydration;
  - relations;
  - QueryBuilder result mapper;
  - `OrmConfiguratorContract` for observers, scopes and morph map.
- DTO package:
  - `DataTransferObject`;
  - DTO factory, mapper and serializer;
  - DTO collections;
  - metadata parser/repository/cache;
  - attributes `MapFrom`, `MapTo`, `CollectionOf`, `CastWith`.
- Validation package:
  - fluent `Rule` API;
  - validator factory and validator;
  - `ValidatedData`;
  - `Request::validate()`;
  - `FormRequest`;
  - configurable messages, attributes and presence verifier.
- Config-driven bootstrap:
  - providers from `config/app.php`;
  - routes from `config/routing.php`;
  - middleware/server/response defaults from `config/http.php`;
  - exception rendering/reporting from `config/exceptions.php`;
  - Prism, Docs, Hashing, DTO, Validation, Halcyon and QueryBuilder config integration.

## Changed

- `boot/app.php` is now the composition entrypoint for base path and env file locations only.
- `boot/providers.php` is no longer used by the standard skeleton.
- `config/*.php` is the main source for runtime defaults.
- `ConfigRepository::get()` now treats explicit `null` config values as valid values.

## Upgrade Notes

Move application providers from `boot/providers.php` to `config/app.php`:

```php
use App\Providers\AppServiceProvider;

return [
    'providers' => [
        AppServiceProvider::class,
    ],
];
```

Move route file configuration to `config/routing.php`:

```php
return [
    'files' => [
        'web' => APP_ROOT.'/routes/web.php',
        'api' => APP_ROOT.'/routes/api.php',
    ],
];
```

Move middleware defaults to `config/http.php`:

```php
return [
    'middleware' => [
        'global' => [],
        'web' => [],
        'api' => [],
    ],
];
```

Delete `boot/providers.php` after migrating its contents.
